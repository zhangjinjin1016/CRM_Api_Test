## CRM 接口自动化测试项目
基于 Postman + Newman 完成 CRM 系统接口自动化测试练习，覆盖核心业务接口流程。

## 项目介绍
CRM系统包含合同管理、课程管理两大业务模块。本项目针对系统接口开展自动化测试实践，搭建简易自动化测试方案，完成接口用例编写、调试与批量执行，用于学习接口自动化相关技术，提升回归测试效率。

## 技术栈
Postman、Newman、JavaScript

## 项目内容
1. 梳理合同管理、课程管理接口业务流程，完成全流程接口自动化用例设计与编写，覆盖新增、查询、修改、删除等常用场景。
2. 使用环境变量统一管理接口地址、登录凭证等配置，实现多环境灵活切换。
3. 通过脚本实现登录凭证自动提取与接口关联，同时添加状态码、业务码、响应字段等多维度断言，校验接口返回结果。
4. 基于 Newman 命令行完成批量自动化执行，生成 HTML 格式测试报告，记录执行结果与异常信息。
5. 调试并解决接口字段不匹配、响应格式异常、断言失败等问题，保证用例稳定运行。

## 测试成果
合同管理模块接口执行通过率 100%，单条业务流程执行时长约 8 秒。自动化执行替代重复手动测试，有效提升测试效率。

## 补充说明
本项目为个人学习练习项目，仅用于接口自动化技术学习与实操。
